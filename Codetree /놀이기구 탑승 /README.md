### 문제 설명

크기가 `N × N`인 격자 형태의 놀이기구가 있다.

총 `N²`명의 학생이 주어진 순서대로 한 명씩 자리에 앉는다. 각 학생은 자신이 좋아하는 학생 4명의 정보를 가지고 있으며, 다음 우선순위에 따라 자리를 선택한다.

1. 상하좌우로 인접한 칸 중 좋아하는 학생이 가장 많은 자리
2. 좋아하는 학생 수가 같다면 인접한 빈칸이 가장 많은 자리
3. 위 조건까지 같다면 행 번호가 가장 작은 자리
4. 행 번호까지 같다면 열 번호가 가장 작은 자리

모든 학생이 자리에 앉은 후, 각 학생 주변에 좋아하는 학생이 몇 명 있는지에 따라 만족도를 계산한다.

| 인접한 좋아하는 학생 수 |  만족도 |
| :-----------: | ---: |
|       0       |    0 |
|       1       |    1 |
|       2       |   10 |
|       3       |  100 |
|       4       | 1000 |

모든 학생의 만족도 합을 구하는 프로그램을 작성한다.

### 입력

첫째 줄에 격자의 크기 `N`이 주어진다.

다음 `N²`개의 줄에는 각 학생의 정보가 주어진다.

각 줄에는 학생 번호와 해당 학생이 좋아하는 학생 4명의 번호가 공백으로 구분되어 주어진다.

학생들은 입력으로 주어진 순서대로 자리에 앉는다.

### 출력

모든 학생을 배치한 후 전체 학생의 만족도 합을 출력한다.
